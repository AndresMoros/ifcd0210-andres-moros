## Resumen de la Ficha Técnica

### Identificación del Certificado de Profesionalidad

- **Denominación**: Desarrollo de aplicaciones con tecnologías web
- **Código**: IFCD0210
- **Familia profesional**: Informática y Comunicaciones
- **Área profesional**: Desarrollo
- **Nivel de cualificación profesional**: 3

#### Unidades de Competencia

1. **UC0491_3**: Desarrollar elementos software en el entorno cliente.
2. **UC0492_3**: Desarrollar elementos software en el entorno servidor.
3. **UC0493_3**: Implementar, verificar y documentar aplicaciones web en entornos internet, intranet y extranet.

#### Competencia General

Desarrollar documentos y componentes software que constituyan aplicaciones informáticas en entornos distribuidos utilizando tecnologías web, partiendo de un diseño técnico ya elaborado, realizando, además, la verificación, documentación e implantación de los mismos.

### Entorno Profesional

- **Ámbito profesional**: Empresas o entidades públicas o privadas de cualquier tamaño que disponen de infraestructura de redes intranet, internet o extranet.
- **Sectores productivos**: Principalmente en el sector servicios, pero presente en todos los sectores productivos.
- **Ocupaciones relacionadas**: Programadores de aplicaciones informáticas, Técnicos de la web, Programador web, Programador multimedia.

### Duración de la Formación

- **Total**: 590 horas

#### Módulos Formativos

4. **MF0491_3**: Programación web en el entorno cliente (180 horas)
   - UF1841: Elaboración de documentos web mediante lenguajes de marcas (60 horas)
   - UF1842: Desarrollo y reutilización de componentes software y multimedia mediante lenguajes de guión (90 horas)
   - UF1843: Aplicación de técnicas de usabilidad y accesibilidad en el entorno cliente (30 horas)

5. **MF0492_3**: Programación web en el entorno servidor (240 horas)
   - UF1844: Desarrollo de aplicaciones web en el entorno servidor (90 horas)
   - UF1845: Acceso a datos en aplicaciones web del entorno servidor (90 horas)
   - UF1846: Desarrollo de aplicaciones web distribuidas (60 horas)

6. **MF0493_3**: Implantación de aplicaciones web en entornos internet, intranet y extranet (90 horas)

7. **MP0391**: Módulo de prácticas profesionales no laborales de desarrollo de aplicaciones con tecnología web (80 horas)

### Perfil Profesional

#### Unidad de Competencia 1: Desarrollar elementos software en el entorno cliente

- **Realizaciones profesionales**:
  - Desarrollar documentos estáticos y dinámicos.
  - Desarrollar componentes software en el entorno cliente.
  - Desarrollar componentes multimedia.
  - Utilizar componentes software ya desarrollados.

#### Unidad de Competencia 2: Desarrollar elementos software en el entorno servidor

- **Realizaciones profesionales**:
  - Desarrollar componentes software en el entorno servidor.
  - Manipular interfaces de acceso a bases de datos.
  - Utilizar servicios distribuidos en otros entornos.

#### Unidad de Competencia 3: Implementar, verificar y documentar aplicaciones web

- **Realizaciones profesionales**:
  - Entregar y distribuir la aplicación web.
  - Elaborar y mantener la documentación de la aplicación web.
  - Realizar pruebas para verificar el correcto funcionamiento de los elementos software.

### Formación del Certificado de Profesionalidad

#### Módulo Formativo 1: Programación web en el entorno cliente

- **Unidad Formativa 1 (UF1841)**: Elaboración de documentos web mediante lenguajes de marcas.
- **Unidad Formativa 2 (UF1842)**: Desarrollo y reutilización de componentes software y multimedia mediante lenguajes de guión.
- **Unidad Formativa 3 (UF1843)**: Aplicación de técnicas de usabilidad y accesibilidad en el entorno cliente.

#### Módulo Formativo 2: Programación web en el entorno servidor

- **Unidad Formativa 1 (UF1844)**: Desarrollo de aplicaciones web en el entorno servidor.
- **Unidad Formativa 2 (UF1845)**: Acceso a datos en aplicaciones web del entorno servidor.
- **Unidad Formativa 3 (UF1846)**: Desarrollo de aplicaciones web distribuidas.

#### Módulo Formativo 3: Implantación de aplicaciones web en entornos internet, intranet y extranet

- **Capacidades y criterios de evaluación**:
  - Establecer los procesos de instalación y distribución de la aplicación.
  - Elaborar y mantener la documentación de la aplicación web.
  - Seleccionar y emplear métodos y juegos de pruebas para verificar las funcionalidades y las especificaciones de rendimiento de la aplicación web.

### Módulo de Prácticas Profesionales No Laborales

- **Código**: MP0391
- **Duración**: 80 horas
- **Capacidades y criterios de evaluación**:
  - Participar en el análisis, diseño y desarrollo de sistemas realizados con tecnologías web.
  - Colaborar en la evaluación de la usabilidad y accesibilidad de los sistemas web.
  - Participar en los procesos de trabajo de la empresa, siguiendo las normas e instrucciones establecidas en el centro de trabajo.
